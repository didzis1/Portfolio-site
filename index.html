COVID-19 tracker

Covid-19 worldwide tracker is a project where people can look up the statistics of corona cases/deaths/recoveries all around the world. You can search the statistics for any specific country and display the information in numbers and in a graph.

DinoDiary

DinoDiary is an application where users can write their own personal diary online. Users can create, edit and delete their own diary posts. Each diary post can have a heading, text and an optional image.

GitHub Demo Video

Tuna House

Tuna House is an imaginary sushi restaurant website where customers can add sushi to their shopping carts. Sushi can be filtered by their type (nigiri, uramaki, maki).
